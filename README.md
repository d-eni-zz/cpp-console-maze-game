# C++ Console Maze Game

A simple console-based mini game developed in C++.

## Description
In this game, the player moves through a maze and tries to escape from a ghost. The game tracks how long the player survives.

## Features
- Player movement (W, A, S, D)
- Ghost chasing logic
- Timer system
- Restart and quit options

## Technologies
- C++
- Visual Studio

## Controls
- W → Up
- A → Left
- S → Down
- D → Right
- R → Restart
- Q → Quit
